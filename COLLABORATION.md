## 🛡️ Project Rules (CRITICAL)
*These must be followed strictly to ensure consistency.*

- **Localization:** ALWAYS use Tunisian Dinar (TND) for prices.
- **Language:** UI must support French (Primary) and Arabic (RTL).
- **Phone Numbers:** Validate for Tunisian format (+216 XX XXX XXX).
- **Styling:** Use `NativeWind` (Tailwind for React Native) where applicable, and Tailwind CSS for Web.
- **Database:** Use Supabase. **NEVER drop the `Auctions` table.** Use soft-deletes (`is_deleted = true`).
